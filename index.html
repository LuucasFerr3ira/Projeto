História dos computadores

Resposta: História dos Computadores

O que são Clients?

Resposta: Cliente é quando você consome a informação de algum lugar baseado em web na internet. O client, ele é o dispositivo que está consumindo aquela informação na internet, que vem de algum server.

O que são Servers?

Resposta: Servidor é um computador/software, com sistema de computação centralizada que fornece serviços a uma rede de computadores, que são os clientes^.

Como é uma estrutura básica de um HTML?

Resposta: A estrutura básica de um HTML é formada pelas tags html, head e body e pelo !doctype (não é uma tag html). É claro que necessita de mais tags para que a página funcione, porém os principais são esses.
